# Интернет-магазин на React + Express

Полноценное веб-приложение интернет-магазина с бэкендом на Express и фронтендом на React.

## 🚀 Технологии

### Бэкенд
- Node.js
- Express.js
- CORS
- nanoid (генерация ID)

### Фронтенд
- React 18
- Axios (HTTP клиент)
- SCSS (стилизация)

## 📋 Функциональность

- ✅ Просмотр всех товаров
- ✅ Фильтрация по категории, цене, наличию
- ✅ Сортировка товаров
- ✅ Добавление новых товаров
- ✅ Редактирование товаров
- ✅ Удаление товаров
- ✅ Статистика магазина
